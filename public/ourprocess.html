Below is an overview of how Independent Call Auditors (ICA) processes any set of audio files for our clients:

  1. Independent Call Auditors securely retrieves all audio files from a designated storage location using appropriate credentials.
  2. All files are stored in a dedicated client share on AES 256-bit encrypted storage.
  3. ICA then scans each file for the presence or absence of client-selected key words or phrases.
  4. ICA then generates a scoring report with a call-quality score for each call.
  5. The scoring report is delivered to the client on an agreeable daily or weekly schedule.
  6. The scoring report presents the client with links to calls for selective review.
  7. All files are deleted from the ICA server after 30 days.
